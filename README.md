# Sentiment-Analysis-using-BERT
BERT-based Sentiment Analysis for Amazon Reviews using Hugging Face Transformers. The project classifies customer reviews into sentiment ratings by leveraging a pre-trained multilingual BERT model for accurate NLP-based sentiment prediction.

### Project Overview

This project performs sentiment analysis on Amazon customer reviews using a pre-trained BERT (Bidirectional Encoder Representations from Transformers) model from Hugging Face. It predicts the sentiment of a review by classifying it into a star rating (1–5), where lower ratings indicate negative sentiment and higher ratings indicate positive sentiment.

### Project Workflow
Load the Amazon Reviews dataset.
Preprocess and clean review text.
Tokenize text using the BERT tokenizer.
Use a pre-trained multilingual BERT model for sentiment prediction.
Classify reviews into 1–5 star sentiment categories.
Evaluate model performance using metrics such as Accuracy and Classification Report.
